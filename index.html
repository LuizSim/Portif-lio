HTML

HTML (HyperText Markup Language) é usado para criar e estruturar páginas web, definindo elementos como texto, imagens e links.

CSS

CSS (Cascading Style Sheets) é responsável por estilizar páginas web, aplicando cores, fontes, layouts e designs responsivos.

JavaScript

JavaScript, linguagem de programação que adiciona interatividade e dinamismo às páginas, como animações e respostas em tempo real.

C++

O C++ é uma linguagem de alto desempenho, ideal para sistemas complexos, motores de jogos e aplicações que exigem controle detalhado de hardware.

Python

Python, uma linguagem versátil e de fácil aprendizado, usada em automação, inteligência artificial, análise de dados e desenvolvimento de aplicações.

TypeScript

TypeScript, uma extensão do JavaScript que adiciona tipagem estática, tornando o desenvolvimento mais seguro e organizado.

React

React é uma biblioteca JavaScript para criar interfaces de usuário, focada em componentes reutilizáveis e performance eficiente.

MySQL

MySQL é um sistema de banco de dados relacional amplamente utilizado, conhecido por sua alta performance, sendo ideal para aplicações web e corporativas.
